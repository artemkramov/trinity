B. Kachinskiy, N. V. Agarkova</em></div><br/><div id="articleAbstract"><h4>Abstract</h4><br/><div>In this work applications of structural analysis methods for hydraulic structures (HS) system security were reviewed. The basic principles of security connectivity model creation were presented with help of the Q-analysis on the examples of two sets: the set of threats and the set of security actions. The connectedness of elements of this system was investigated, numerical values of eccentricities and p-holes were calculated, and the degree of complexity of security system elements was analyzed. If a system property of security is taken into account, it can be concluded that the set of threats and the set of security actions are connected and create the basis for the security system.</div><br/></div><div id="articleFullText"><h4>Full Text:</h4><a href="http://journal.iasa.kpi.ua/article/view/53413/49445" class="file" target="_parent">PDF (Русский)</a></div><div id="articleCitations"><h4>References</h4><br/><div><p>Malik L.K.
