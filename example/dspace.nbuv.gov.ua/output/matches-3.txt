Синтез многоуровневых структур со многими выходами">Синтез многоуровневых структур со многими выходами
<div class="spacer"></div>В работе рассматривается метод решения задачи адаптации логической сети со многими выходами с восстановлением входного множества двоичных векторов при заданных только младших значениях этих векторов и значениях на выходах сети. Алгоритм синтеза логической сети основан на описании ее полиномом Жегалкина.<div class="spacer"></div>В роботі розглядається метод розв’язання задачі адаптації логікової мережі з багатьма виходами з відновленням вхідної множини двійкових векторів при заданих тільки молодших значеннях цих векторів і значень на виходах мережі. Алгоритм синтезу логікової мережі грунтується на зображенні її поліномом Жегалкіна.<div class="spacer"></div>The method for solution of adaptation problem of the logical network with many outputs for the restoration of the input set of binary vectors when given only the lower values of this set and the values of the outputs is considered. The algorithm synthesis of the logical network is based on the description of its polynomial Zhegalkin.<div class="spacer"></div>
